PluginInterface¶

Interface for self-registering event listeners.

Classes implementing this interface may be registered by name or instance with an EventManager, without an event name. The \Zend\Cache\Storage\Plugin\attach() method will then be called with the current EventManager instance, allowing the class to wire up one or more listeners.
