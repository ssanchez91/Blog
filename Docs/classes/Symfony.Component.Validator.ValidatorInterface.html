Validates a value.

validate(mixed $value, array|null $groups = null, boolean $traverse = false, boolean $deep = false) : \Symfony\Component\Validator\ConstraintViolationListInterface

The accepted values depend on the MetadataFactoryInterface implementation.

The signature changed with Symfony 2.5 (see Validator\ValidatorInterface::validate(). This signature will be disabled in Symfony 3.0.

Arguments

$value

mixed

The value to validate

$groups

array|null

The validation groups to validate

$traverse

boolean

Whether to traverse the value if it is traversable

$deep

boolean

Whether to traverse nested traversable values recursively

Response

\Symfony\Component\Validator\ConstraintViolationListInterface

A list of constraint violations. If the list is empty, validation succeeded.
