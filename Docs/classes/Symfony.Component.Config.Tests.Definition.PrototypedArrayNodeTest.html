Tests that when a key attribute is mapped, that key is removed from the array.

testMappedAttributeKeyIsRemoved()

The above should finally be mapped to an array that looks like this (because "id" is the key attribute).

array(
    'things' => array(
        'option1' => 'foo',
        'option2' => 'bar',
    )
)

Tests that when a key attribute is mapped, that key is removed from the array.

testMappedAttributeKeyIsRemovedLeftValueOnly( $value,  $children,  $expected)

And if only 'value' element is left in the array, it will replace its wrapper array.

<things>
    <option id="option1" value="value1">
</things>

The above should finally be mapped to an array that looks like this (because "id" is the key attribute).

array(
    'things' => array(
        'option1' => 'value1'
    )
)

It's also possible to mix 'value-only' and 'non-value-only' elements in the array.

The above should finally be mapped to an array as follows

array( 'things' => array( 'option1' => 'value1', 'option2' => array( 'value' => 'value2', 'foo' => 'foo2' ) ) )
