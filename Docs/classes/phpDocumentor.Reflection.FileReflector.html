Opens the file and retrieves its contents.

__construct(string $file, boolean $validate = false, string $encoding = 'utf-8')

During construction the given file is checked whether it is readable and if the $validate argument is true a PHP Lint action is executed to check whether the there are no parse errors.

By default the Lint check is disabled because of the performance hit introduced by this action.

If the validation checks out, the file's contents are read, converted to UTF-8 and the object is created from those contents.

Throws
\phpDocumentor\Reflection\Exception\UnreadableFile

If the filename is incorrect or the file cannot be opened

\phpDocumentor\Reflection\Exception\UnparsableFile

If the file fails PHP lint checking (this can only happen when $validate is set to true)

Checks whether the given node is recogized by phpDocumentor as a documentable element.

isNodeDocumentable(\PhpParser\Node $node) : boolean

The following elements are recognized:

  • Trait
  • Class
  • Interface
  • Class constant
  • Class method
  • Property
  • Include/Require
  • Constant, both const and define
  • Function

leaveNode(\PhpParser\Node $node) : null|\PhpParser\Node|false|array<mixed,\PhpParser\Node>

Return value semantics:

  • null: $node stays as-is
  • false: $node is removed from the parent array
  • array: The return value is merged into the parent array (at the position of the $node)
  • otherwise: $node is set to the return value
