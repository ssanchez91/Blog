TestMemberMetadata

Extends \Symfony\Component\Validator\Mapping\MemberMetadata

Stores all metadata needed for validating a class property.

The method of accessing the property's value must be specified by subclasses by implementing the \Symfony\Component\Validator\Tests\Mapping\newReflectionMember() method.

This class supports serialization and cloning.

Adds a constraint.

addConstraint(\Symfony\Component\Validator\Constraint $constraint) : $this
inherited

If the constraint Valid is added, the cascading strategy will be changed to CascadingStrategy::CASCADE. Depending on the properties $traverse and $deep of that constraint, the traversal strategy will be set to one of the following:

  • TraversalStrategy::IMPLICIT if $traverse is enabled and $deep is enabled
  • TraversalStrategy::IMPLICIT | \Symfony\Component\Validator\Mapping\TraversalStrategy::STOP_RECURSION if $traverse is enabled, but $deep is disabled
  • TraversalStrategy::NONE if $traverse is disabled
