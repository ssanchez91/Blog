Create document from docbook document

createFromDocbook(\ezcDocumentDocbook $document) : void
inherited abstract

A document of the docbook format is provided and the internal document structure should be created out of this.

This method is required for all formats to have one central format, so that each format can be compiled into each other format using docbook as an intermediate format.

You may of course just call an existing converter for this conversion.

Arguments

$document

\ezcDocumentDocbook

Return document compiled to the docbook format

getAsDocbook() : \ezcDocumentDocbook
inherited abstract

The internal document structure is compiled to the docbook format and the resulting docbook document is returned.

This method is required for all formats to have one central format, so that each format can be compiled into each other format using docbook as an intermediate format.

You may of course just call an existing converter for this conversion.

Set DOMDocument

setDomDocument(\DOMDocument $document) : void
inherited

Directly set the internally stored DOMDocument object, to spare additional XML parsing overhead. Setting a broken or invalid docbook document is not checked here, ebcause validation would cost too much performace on each set. Be careful what you set here, invalid documents may lead to unpredictable errors.
