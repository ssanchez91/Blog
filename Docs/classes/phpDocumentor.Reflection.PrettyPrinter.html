PrettyPrinter

Extends \PhpParser\PrettyPrinter\Standard

Custom PrettyPrinter for phpDocumentor.

phpDocumentor has a custom PrettyPrinter for PHP-Parser because it needs the unmodified value for Scalar variables instead of an interpreted version.

If the interpreted version was to be used then the XML interpretation would fail because of special characters.
