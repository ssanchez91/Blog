ezcDocumentOdtTextProcessor

Processes text nodes with significant whitespaces.

An instance of this class is used to process DOMText nodes with significant whitespaces (such as literallayout).

getMatchType(string $string) : string

Returns a string indicating what type of whitespaces has been matched. This string is also the name of the text:* tag used to reflect the whitespace in ODT:

  • 's' for spaces
  • 'tab' for tabs
  • 'line-break' for line breaks

Checks if whitespaces need additional processing and returns the corresponding DOMText for the ODT.

processText(\DOMText $textNode, \DOMElement $newRoot) : \array(DOMNode)

This method checks if the given $textNode is descendant of a tag. In this case, whitespaces are processed according to the ODT specs:

  • More than 2 simple spaces are replaced by a single space and <text:s /> with the text:c attribute set to the number of spaces - 1.
  • One or more tabs / line breaks are replaced by a / tag, with the text:c attribute set to the number of whitespaces replaced.
