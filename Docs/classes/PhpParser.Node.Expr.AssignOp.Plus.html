__construct(null|array $subNodes = array(), array $attributes = array())
inherited

If null is passed for the $subNodes parameter the node constructor must assign all subnodes by itself and also override the getSubNodeNames() method. DEPRECATED: If an array is passed as $subNodes instead, the properties corresponding to the array keys will be set and getSubNodeNames() will return the keys of that array.
