JSON encode an array value.

encodeArray( $array) : string

Recursively encodes each value of an array and returns a JSON encoded array string.

Arrays are defined as integer-indexed arrays starting at index 0, where the last index is (count($array) -1); any deviation from that is considered an associative array, and will be passed to \Zend\Json\encodeAssociativeArray().

Arguments

$array

array

Response

string

encodeAssociativeArray(array $array) : string

Encode an object to JSON by encoding each of the public properties.

encodeObject( &$value) : string

A special property is added to the JSON object called '__className' that contains the classname of $value; this can be used by consumers of the resulting JSON to cast to the specific class.

Encode a value to JSON.

encodeValue( &$value) : string

Recursive method which determines the type of value to be encoded and then dispatches to the appropriate method.

$values are either

  • objects (returns from \Zend\Json\encodeObject())
  • arrays (returns from \Zend\Json\encodeArray())
  • scalars (returns from \Zend\Json\encodeDatum())
