Set

Implements \PhpCollection\SetInterface

Implementation of a Set.

Each set guarantees that equal elements are only contained once in the Set.

This implementation constraints Sets to either consist of objects that implement ObjectBasics, or objects that have an external ObjectBasicsHandler implementation, or simple scalars. These types cannot be mixed within the same Set.
