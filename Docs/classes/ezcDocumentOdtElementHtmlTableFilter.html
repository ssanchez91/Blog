ezcDocumentOdtElementHtmlTableFilter

Extends \ezcDocumentOdtElementBaseFilter

Filter for ODT <table:table/> and its child elements.

This filter generates XHTML style tables in DocBook. This class is not yet in use at all, since the internal DocBook representation of the Document component does not cope with XHTML style tables, yet.
