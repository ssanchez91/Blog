ezcDocumentOdtListStyleGenerator

Extends \ezcDocumentOdtStyleGenerator

Class to generate styles for lists (<text:list/>).

Style generators used in ezcDocumentOdtStyler must extend this abstract class.

Calculates the list margin and indent.

calculateListLevelMeasures(\DOMElement $listStyle, integer $level, array $styleAttributes) : \array(string=>float)

Margin and indent are handled in a strange way in ODF. This method calculates the margin for a list level, based on the previous level margin and the current margin and padding. In addition, the text-indent is set to fit the previous list-level. The new $styleAttributes are returned.

createListStyle(\ezcDocumentOdtStyleInformation $styleInfo, \DOMElement $list, array $styleAttributes) : void

Checks if the list is nested in a different list. If this is not the case, a new list style is generated. Otherwise, the existing list style is retrieved and a list definition for the corresponding nesting depth is created.
