ezcDocumentOdtFormattingPropertyCollection

Class to carry and manage {@link ezcDocumentOdtFormattingProperties}.

An instance of this class is used in an ezcDocumentOdtStyle to carry various formatting properties of class {@link ezcDocumentOdtFormattingProperties}.

Returns properties of the given $type.

getProperties(\const $type) : \ezcDocumentOdtFormattingProperties|null

If properties of the given $type are set, the corresponding object is returned. Otherwise null is returned. You can check if properties of a given $type are set using \hasProperties(). $type must be one of the ezcDocumentOdtFormattingProperties FAMILY_ constants.
