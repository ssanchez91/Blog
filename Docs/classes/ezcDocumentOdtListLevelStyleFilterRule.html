ezcDocumentOdtListLevelStyleFilterRule

Implements \ezcDocumentOdtStyleFilterRule

Style filter rule to distinguish list types.

ODT does not distinguish between numbered and itemized lists on an XML element level, but through styling information. This rule implements detection of numbered lists.
